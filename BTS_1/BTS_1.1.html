1. A cosa serve bootstrap?

Bootstrap serve a sviluppare velocemente siti web e applicazioni responsive e "mobile-first" grazie a una raccolta di componenti e strumenti predefiniti (CSS e JavaScript). Aiuta a creare layout, pulsanti, navigazione e moduli standard, risparmiando tempo e garantendo la compatibilità tra diversi browser e dispositivi.

2. Cosa devi importare per creare una pagina web con bootstrap

Per creare una pagina web con Bootstrap, devi importare i file CSS e JavaScript del framework. Puoi farlo scaricando i file e collegandoli nel tuo HTML, oppure utilizzando una rete di distribuzione dei contenuti (CDN) per caricarli da un server remoto. per scaricare devi tramite remoto il link <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

3. Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Un CDN (Content Delivery Network) è una rete di server distribuiti geograficamente che memorizzano copie (cache) di file statici come immagini, video, fogli di stile (CSS), script JavaScript, ecc., per poi fornirli agli utenti dal server più vicino a loro.
